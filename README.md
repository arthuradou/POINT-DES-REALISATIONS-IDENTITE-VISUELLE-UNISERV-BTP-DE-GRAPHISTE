POINT-DES-REALISATIONS-IDENTITE-VISUELLE-UNISERV-BTP-DE-GRAPHISTE Description

Interface web interactive permettant de suivre l’avancement des points de réalisations de l’identité visuelle de UNISERV BTP et sa communication sur les réseaux sociaux.

Le projet présente les différentes phases, les livrables associés et leur statut en temps réel.

Fonctionnalités

Suivi des tâches par statut : Validé, En cours, Attention, Incomplet Organisation en phases (fondamentaux, templates, autres livrables) Barre de progression automatique Interface responsive (mobile et desktop) Interaction simple via boutons

Technologies

HTML5 CSS3 JavaScript

Utilisation

Ouvrir index.html dans un navigateur Mettre à jour les statuts en cliquant sur les boutons Suivre la progression globale automatiquement Mise en ligne

Projet réalisé dans le cadre du suivi du plan de communication de UNISERV BTP.
